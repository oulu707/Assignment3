# Instructions:
Django was used in the creation of the portal and all relevant files are included in the .zip folder.

Instructions for running this program:

Install django in the python terminal https://youtu.be/UmljXZIypDc

Install crispy forms with this code

pip install django-crispy-forms

Run migrations with this code
python manage.py migrate

Run the code
python manage.py runserver

The URL will be shown in the terminal
